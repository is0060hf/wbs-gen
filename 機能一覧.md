# WBS管理システム 機能一覧プロンプト

## 基本機能

### 1. プロジェクト管理
```
- [ ] 新規プロジェクト作成（空のテンプレートから開始）
- [ ] プロジェクト情報編集（名前、説明、バージョン）
- [ ] サンプルプロジェクト生成（デモ用データ）
```

### 2. ファイル操作
```
- [ ] JSONファイルインポート（ファイル選択ダイアログ）
- [ ] JSONテキストインポート（テキストエリアからペースト）
- [ ] JSONファイルエクスポート（ダウンロード）
- [ ] JSONクリップボードコピー
- [ ] インポート時のデータ検証とエラー表示
- [ ] 不完全データの自動補完
```

### 3. タスク管理
```
- [ ] タスク新規作成（ルートレベル）
- [ ] 子タスク追加（選択タスクの配下）
- [ ] 兄弟タスク追加（選択タスクの後）
- [ ] タスク削除（確認ダイアログ付き）
- [ ] タスク複製
- [ ] タスク一括選択
- [ ] タスク一括削除
- [ ] タスク一括編集
```

### 4. タスク編集
```
- [ ] タスク名インライン編集（ダブルクリック）
- [ ] タスク詳細モーダル編集
- [ ] 開始日の変更（カレンダーピッカー）
- [ ] 期間の変更（日数入力）
- [ ] 終了日の自動計算
- [ ] 進捗率の更新（0-100%）
- [ ] ステータスの変更（未着手/進行中/完了/遅延）
- [ ] 優先度の設定（Must/Should/Could/Won't）
- [ ] 担当者の設定
- [ ] 説明文の追加・編集
- [ ] メモの追加・編集
```

## 実装優先度の推奨

### 必須機能（MVP）
1. 基本的なタスク管理（作成・編集・削除）
2. JSONインポート/エクスポート
3. ガントチャートまたはテーブルビュー（どちらか一方）
4. 依存関係の基本管理
5. ドラッグ&ドロップによる並び替え

### 推奨機能（Phase 2）
1. 両方のビュー実装
2. フィルタリング・検索
3. ショートカットキー
4. 進捗管理
5. クリティカルパス表示

### 拡張機能（Phase 3）
1. 一括編集
2. 高度なフィルタリング
3. 操作履歴（Undo/Redo）
4. パフォーマンス最適化
5. カスタマイズ機能

## 使用例

開発時には、以下のようにプロンプトで機能を指定できます：

```
以下の機能を実装してください：
- 基本機能: 1, 2, 3, 4
- 高度な機能: 5（前提タスクの追加・削除のみ）, 6（クリティカルパス識別のみ）
- ビュー機能: 8（基本的なガントチャート）, 9（基本的なテーブル）
- フィルタリング: 10（タスク名検索と優先度フィルターのみ）
- ショートカット: 11（Ctrl+S, Ctrl+O, Delete のみ）
```

このように、必要な機能を選択的に実装指定することで、プロジェクトの要件に応じた柔軟な開発が可能です。
